##### Make sure: 

###### There are 4 different fonts

###### There are max 2 different typographies (Bold, italic, normal, light, etc. are fonts, not typography)
###### Each moodboard should have typography:
    ###### 1 for a title
    ###### 1 for a button
    ###### 1 for a description
    ###### 1 for the text
